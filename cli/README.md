# QSO Logger CLI

This directory contains the command-line interface (CLI) for the QSO Logger application. The CLI allows users to interact with the QSO Logger functionalities directly from the terminal.

## Installation

To install the CLI, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/h2-technologies/qso-logger.git
   ```

   or if you are using your own fork:

   ```bash
   git clone https://github.com/yourusername/qso-logger.git
   ```

2. Navigate to the CLI directory:

   ```bash
   cd qso-logger/cli
   ```
